# Recruit UI Kit

Recreation of the dept recruit deck (source: `github.com/da33/lin-yu-fan/fashion-recruit.html`).

## Components
- `DeckSlide` — full-viewport slide shell with background image + 72% dark overlay, numbered label
- `DeckH1`, `DeckH2`, `DeckSub`, `DeckLine` — gold type primitives
- `CardGrid` — N-column hairline-grid of recruit cards (tag/title/desc)
- `PathList` — two-column rows (left: number+name, right: tagline+description)
- `DeckNav` — prev/next arrows, bottom progress fill, slide counter

## Philosophy
Black ground. **Gold `#c9a84c`** is the ONLY accent (department/official register). Bilingual with Chinese-first headlines. Numbered eyebrows `01 · VISION`. Persistent slide index in localStorage. Arrow-key navigation.

## Slides included
Cover → Vision (六條出路) → Directions (四個方向) → Value (錢的問題) → Korea Study Tour.
