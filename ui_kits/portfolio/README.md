# Portfolio UI Kit

Recreation of Rino's personal portfolio (source: `github.com/da33/lin-yu-fan/index.html`).

## Components
- `Nav` — fixed top nav, mix-blend-difference, nav link list
- `Hero` — full-viewport, two-line wordmark (solid + outline), bottom-aligned content
- `Marquee` — horizontal infinite scroll with ✦ separators (flare-colored)
- `About` — two-column 001 section, TC body with paper-inline strong, skill chips
- `Experience` — numbered list rows with padding-left hover shift
- `Global` — 2×2 hairline-grid of country cards with flag emoji + TC description
- `Contact` — underlined input + full-width CTA row; sent confirmation state

## Philosophy
Dark editorial. Sharp corners. Hairline dividers, no shadows. Every section opens with a 3-digit numbered eyebrow (`001 — ABOUT`). Bilingual copy — English headlines, TC body. One accent color (`--flare`).
